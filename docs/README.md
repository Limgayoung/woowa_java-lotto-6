## 주의사항
- 메서드 15 line 이하로 작성
- else 사용 금지
- enum 사용
- 핵심 로직과 ui 로직 분리 구현

## 기능 목록
- [ ] 입력 
   - [ ] 구입 금액 입력
   - [ ] 로또 당첨 번호 입력(,로 구분)
   - [ ] 보너스 번호 입력
- [ ] 출력
   - [ ] 발행 수량 출력
   - [ ] 로또 번호 오름차순 정렬 후 출력 
   - [ ] 당첨 내역 출력
   - [ ] 수익률 출력
- [ ] 로또
   - [ ] 로또 발행 (랜덤 숫자 6개 뽑기)
   - [ ] 당첨 확인
   - [ ] 수익률 계산
- [ ] 예외 상황
